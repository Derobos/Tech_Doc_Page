Linux is an open source operating system. This means that it is free to use, modify, or distribute. That offers a lot of freedom for users, programmers, and developers to customize their operating system and much of the software used. There are many additional benefits to an open source operating system such as less preloaded "bloatware" and more privacy. This can all be confusing for new users. There are many guides and other help can be found online within the Linux community.

Choose a version
  1. Determine the specifications for the computer you want to use.
  2. Make note of the CPU, amount of RAM (memory), and storage (HDD, SSD, or flash drive) available.
  3. Research which distributions will work best for your computer
  4. You may also be able to choose from a selection of desktop styles.

This guide assumes you want to install linux on a desktop or laptop computer. It may also be useful if you are installing Linux on a flashdrive or virtual machine. Become familiar with the target hardware you want to intall Linux on.

Key things to identify are the CPU (32 bit vs 64 bit), amount of RAM, and available storage space. These factors may help you decide which Linux distribution you want to install.

Look around at the different distributions of Linux that are currently available. The most popular are based on either Debian or Red Hat. Choosing a more popular version will make software and guides easier to find.

Get an iso
  1. Download the version of Linux you have chosen.
  2. Choose the right speed (32 bit vs. 64 bit) for your CPU.
  3. Save the iso to your HDD or SSD drive.
  4. When your iso is finished downloading burn it to a USB thumb drive or DVD

Once you've settled on a version of Linux you would like to try it's time to download the iso image. Most people recommend downloading this file from the distributions official site, the download should be secure and free.

Be sure you choose the right version for the machine you will be installing it on. The 64 bit version will not run on a 32 bit system. Save this download to your HDD or SSD.

Next you'll need to burn the iso image to a DVD or flash drive using the utility software of your choice. Something like "Rufus" or "DeepBurner" there are many options available and you should choose the one you are comfortable with.

Installation
  1. Make sure partitions are set if needed.
  2. restart the machine and boot from the DVD or flash drive.
  3. Select "Try it" to confirm you want to install.
  4. Click install from the desktop.
  5. Be sure you select the right drive to install Linux on.

Some distributions or configurations may need your HDD or SDD to have partitions set up ahead of time. This should all be explained in the written installation instructions for the distribution you've chosen if it is necessary.

Put your bootable DVD or flash drive in the computer and restart your machine. You may need to enter the "Bios" to boot from your DVD or flash drive. This is usually done by pressing "Escape" or one of the "Function" keys during the startup process. If your computer loads windows then you may have to restart it and try again. On windows 10 you can access the bios from the start button under recovery options. From the Bios screen, select the boot menu and select either DVD or USB to boot from the iso you made.

The next step will ask you if you want to try Linux or install it on your computer. It is probably best to try it first, to make sure it is working properly and that you like the feel of it. If you decide it isn't for you, simply unplug the flash drive or cd and restart your machine. Once you find a version you like move on to a full installation on your computer. Be sure to select the right target drive when you are doing a full install, or you may accidentally erase important data.

Set up
  1. Update your system.
  2. Create a restore point.
<>The most important thing to do once you've installed Linux is a system update. This will require an internet connection to download any necessary updates. If you are using Wifi and it did not connect after installation you may need a wired connection to complete this process. You may have a preinstalled update manager, if not you can open up a terminal (ctrl+alt+T) and type (sudo apt update) then press enter. It will ask for your password and then check for updates for your system. Next type (sudo apt upgrade) into the terminal and press enter. The updates to be installed will be listed in the terminal and then ask if you want to proceed. Type y then press enter.

Next you may want to create a system backup in case something goes wrong and you need to restore your system. There may be a utility pre-installed on the version of Linux you have or you may have to install one yourself.

Now you should be safe to explore your new operating system and customize it how you see fit. There are many other guides out there that suggest ways in which you might tweak your environment and software you might find useful. I hope you will take the time to explore some of them and that you have fun on your journey.

Basic terminal commands

A few commands you may frequently use in a Linux terminal, and a short description of what they can do.

(Sudo) Superuser do: allows a user to run programs with the priveleges of another user (superuser by default).

(pwd) Print working directory: Prints the path of the directory you are currently in starting from the root directory.

(cd "directory name") Change directory: Changes the directory you are in.

(cat "filename") Concatenate:Prints the contents of a file in the terminal screen.

(touch "filename") Creates a file in the current directory. If the file already exists, it will update the timestamp on the file.

(man"name") Manual: Opens a manual if one exists.
